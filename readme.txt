# 🤖 Simple Chatbot

## 📌 Description
This is a simple chatbot built using Python and Streamlit.  
It combines rule-based responses with AI-generated replies.

## 🚀 Features
- Basic conversation (hello, bye, etc.)
- AI responses using OpenAI API
- Simple and clean web interface

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API

## ▶️ How to Run

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

## 💡 Approach
- Rule-based system for simple queries
- AI API for dynamic responses

## ⚠️ Challenges
- Managing chat history
- Handling unknown inputs
