# 🤖 Simple Chatbot

## 📌 Description
This is a simple chatbot built using Python and Streamlit.  
It combines rule-based responses with AI-generated replies from OpenRouter API (free alternative to OpenAI).

## ✨ Features
- Basic conversation (hello, bye, jokes, etc.)
- AI-powered responses using OpenRouter API
- Simple and clean web interface
- Chat history management
- Clear chat functionality

## 🛠 Tech Stack
- Python 3.8+
- Streamlit 1.28+
- OpenRouter API (Free tier)
- Requests library
